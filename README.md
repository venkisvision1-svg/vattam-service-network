# VATTAM Service Network Platform

A production-ready multi-service booking platform built with Next.js, Tailwind CSS, and MongoDB Atlas.

## Features

✨ **Services Offered**
- AC Service & Repair
- CCTV Installation
- Fridge Repair
- Washing Machine Repair
- Electrician Services
- Plumbing Services
- Solar Panel Installation & Service
- RO (Reverse Osmosis) Service
- Digital Marketing
- Website Creation

🎯 **Core Features**
- Dark Orange-Black UI Theme
- Customer Booking Portal
- Admin Panel with Analytics
- Technician Dashboard
- QR Business Profiles
- Bilingual Support (Tamil + English)
- Mobile-First Responsive Design
- MongoDB Atlas Integration
- WhatsApp Booking Integration
- Vercel Deployment Ready

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Database**: MongoDB Atlas
- **API**: RESTful with Next.js API Routes
- **Notifications**: WhatsApp API
- **Deployment**: Vercel

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account
- WhatsApp Business API access

### Installation

1. Clone the repository
```bash
git clone https://github.com/venkisvision1-svg/vattam-service-network.git
cd vattam-service-network
```

2. Install dependencies
```bash
npm install
```

3. Setup environment variables
```bash
cp .env.local.example .env.local
```

4. Run development server
```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Deployment

### Vercel Deployment

1. Push to GitHub
2. Import project in Vercel Dashboard
3. Add environment variables
4. Deploy
